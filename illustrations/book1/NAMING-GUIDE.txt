WHISPERWOOD BOOK 1 — ILLUSTRATION DROP FOLDER
==============================================

Drop your finished illustration files here.
Once all (or some) images are in place, tell Claude and the placement
script will copy them into the ePub and print files automatically.

-----------------
Name files exactly as shown. The script matches by filename.

  cover.jpg       Front cover art

  illo-02.jpg     Ch 1 — Mapleaf veins go dark + Pip lands on Vera
  illo-03.jpg     Ch 2 — Sage rings the bellflower at the rising river

FILE FORMAT
-----------
  Format:   JPG or PNG (JPG preferred for final art — smaller file size)
  Color:    RGB (not CMYK — KDP handles conversion)
  Bit depth: 8-bit


SIZE REQUIREMENTS
-----------------
Single-page illustrations (illos 02–05, 07–08, 10–11):
  Minimum for ebook:  1700 × 1700 px  (200 ppi at 8.5")
  Minimum for print:  2625 × 2625 px  (300 ppi at 8.75" with bleed)
  → If generating at print size (2625×2625), the ebook copy will scale down fine.

Full-spread illustrations (illos 01, 06, 09, 12) — TWO PAGES WIDE:
  Minimum for ebook:  3400 × 1700 px  (200 ppi across two 8.5" pages)
  Minimum for print:  5250 × 2625 px  (300 ppi across two 8.75" pages)
  → Same file used for both formats. The print layout crops each half
    onto its own page automatically.

Cover (cover.jpg) — single square page:
  Minimum:  2625 × 2625 px  (300 ppi at 8.75" with bleed)
  Note: KDP also requires a separate cover file for the back cover +
  spine. That is handled via KDP's Cover Creator tool at upload time.
  This file is the FRONT COVER only, used inside the ebook as well.


PARTIAL DROPS ARE FINE
-----------------------
You don't need all 13 files at once. Drop what you have, tell Claude,
and the script will place whatever is available. Missing images stay
as placeholders. Run placement again when more images are ready.


HOW TO TRIGGER PLACEMENT
-------------------------
1. Drop image files here (named exactly as above)
2. Commit and push to git
3. Tell Claude: "images are ready, please place them"
4. Claude runs: production/place-illustrations.sh
5. Both formats update automatically. ePub is rebuilt and ready to upload.
